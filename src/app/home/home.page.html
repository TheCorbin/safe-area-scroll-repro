Safe Area Scroll Repro

Scroll + Keyboard Test

This form has many fields to test scrolling behavior when the keyboard is open. With the @capacitor-community/safe-area plugin installed and EdgeToEdge.enable(this) in the Android activity, the page should scroll when the keyboard is up. If it doesn't, that's the bug.

Steps to reproduce:

  1. Open this app on an Android device
  2. Tap any input field (keyboard opens)
  3. Try to scroll the page while the keyboard is visible
  4. Observe: the page cannot be scrolled
  5. Try to dismiss the keyboard by tapping outside an input
  6. Observe: the keyboard is hard to dismiss
